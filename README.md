# Bright Minds Smart Home Hub

## Overview

This project was developed for the ACCA5036 – Scripting and Programming Concepts module.

The application provides a simple way to manage smart home devices. Users can view all registered devices, search for a specific device, add new devices and update the status of existing devices through a menu-driven interface.

## Features

- View all devices
- Search for a device
- Add a new device
- Update device status
- Device status validation

## Technologies

- Python 3
- Visual Studio Code
- Git
- GitHub

## Programming Concepts Used

- Variables
- Lists
- Dictionaries
- User-defined Functions
- if / elif / else statements
- for loop
- while loop
- Modular programming
